Tutor Teaching Script: 三阶幻方填数 / 3×3 Magic Square

Use this as a spoken script. Each step includes an image cue that can be shown on screen or printed.

Main answer: 2, 9, 4 / 7, 5, 3 / 6, 1, 8
Opening — Set the goal

What are we trying to do?

Say: “Today we are completing a 3×3 magic square. A magic square means every row, every column, and both diagonals must have the same total. We already know three numbers: 9, 4, and 3.”
Ask: “Which places are already filled? Which places are still empty?”
Image cue: Original problem grid with only 9, 4, and 3 filled in.
Known cells: top-middle = 9, top-right = 4, middle-right = 3.
Step 1 — Find the magic sum

Use the 1–9 property

Say: “Because this puzzle uses the numbers 1 through 9, first find their total. The total is 45. Since there are three rows with the same sum, each row must add to 45 ÷ 3, which is 15. So the magic sum is 15.”
Ask: “Why do we divide by 3, not by 9?”
Image cue: Number cards 1–9 flowing into 45, then 45 ÷ 3 = 15.
Every complete line must equal 15.
Step 2 — Fill the top-left corner

Use the first row

Say: “Look at the top row. It has one missing number, plus 9 and 4. The whole row must be 15. So blank + 9 + 4 = 15. Since 9 + 4 = 13, the missing number is 2.”
Image cue: Highlight the top row and write □ + 9 + 4 = 15.
Top-left = 2.
Step 3 — Fill the bottom-right corner

Use the right column

Say: “Now the right column is almost complete: 4, 3, and one blank. It also has to add to 15. So 4 + 3 + blank = 15. Since 4 + 3 = 7, the missing number is 8.”
Image cue: Highlight the right column and write 4 + 3 + □ = 15.
Bottom-right = 8.
Step 4 — Fill the center

Use the main diagonal

Say: “The diagonal from top-left to bottom-right now has 2, blank, and 8. It must also total 15. So 2 + blank + 8 = 15. The missing number is 5. This is a very important moment: in a 1–9 magic square, the center is 5.”
Ask: “What does the diagonal tell us immediately after we know 2 and 8?”
Image cue: Highlight the diagonal 2–5–8.
Center = 5.
Step 5 — Fill the middle-left number

Use the second row

Say: “The second row now has blank, 5, and 3. The row must equal 15. So blank + 5 + 3 = 15. Since 5 + 3 = 8, the blank is 7.”
Image cue: Highlight the middle row.
Middle-left = 7.
Step 6 — Fill the bottom-left number

Use the first column

Say: “Now look at the first column: 2, 7, and blank. This column must also equal 15. So 2 + 7 + blank = 15. Since 2 + 7 = 9, the blank is 6.”
Image cue: Highlight the first column.
Bottom-left = 6.
Step 7 — Fill the last empty number

Use the bottom row

Say: “Only one square is left. The bottom row has 6, blank, and 8. It must equal 15. So 6 + blank + 8 = 15. Since 6 + 8 = 14, the missing number is 1.”
Image cue: Highlight the bottom row.
Bottom-middle = 1.
Step 8 — Verify the answer

Check rows, columns, and diagonals

Say: “We should not stop after filling the grid. We verify it. Each row equals 15, each column equals 15, and both diagonals equal 15. That proves the square is correct.”
Ask: “Can you choose one row, one column, and one diagonal to check?”
Image cue: Final completed grid with eight checks around it.
Final answer: every line equals 15.
Optional extension — Algebra method

For stronger students

Say: “Another way is to let the top-left number be a and the common sum be S. The first row gives S = a + 13. The right column gives the bottom-right number as a + 6. When we combine the equations, we get a = 2, so S = 15.”
Image cue: Label the unknown cells with letters a, d, e, g, h, i and show the equations.
This method is good for algebra practice.
Optional extension — Position relationship

Show a useful pattern

Say: “In a 3×3 magic square, a corner can be related to nearby edge-middle cells. For the top-left corner, we can derive 2a = h + f, so a = (h + f) ÷ 2. Here f is 3 and h is 1, so a = (3 + 1) ÷ 2 = 2.”
Image cue: Highlight a, f, and h, then show a = average of f and h.
A neat extension after students understand the main solution.
Closing — Student recap

Let the student explain back

Say: “Your turn: explain why the magic sum is 15, then explain how we used a row, a column, and a diagonal to finish the square.”
Check understanding: Student should say: total 1–9 is 45, each row sum is 15, then each missing cell is found from one line with only one blank.
PromptExpected student answer
Why is the magic sum 15?1–9 total 45; 3 rows share the total; 45 ÷ 3 = 15.
How did we find 2?□ + 9 + 4 = 15.
How do we know the answer is correct?Rows, columns, and diagonals all add to 15.
